# Security Policy

## Supported Versions

We currently only support the latest version of Mixtura. Please verify that you are running the most recent release before reporting issues.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of Mixtura seriously. If you have found a vulnerability, **please do not open a public issue.**

### How to Report

1. Email us privately at **miguelpinotty@gmail.com** (or use the GitHub Security Advisory "Report a vulnerability").
2. Describe the vulnerability with as much detail as possible.
3. Include steps to reproduce the issue.

### Our Commitment

- We will acknowledge your report within 48 hours.
- We will provide a timeline for investigation and resolution.
- We will openly thank you for your responsible disclosure once the issue is resolved.
